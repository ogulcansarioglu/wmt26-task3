54,419 (segment, system) rows, 14 LPs, global error-free rate **0.134**.

zero annotated error spans` (sensitivity variant `error_free_all` kept).

en-mas_KE (0% positive) as the degenerate-LP case; which WMT26 pairs are
covered (10/23) vs. threshold-transfer pairs (13/23, incl. en-de).
